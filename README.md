# Movie-Cli

A Movie commandline application that uses Omdb Database and Api

# Usage

To use this app you must have node, npm and npx installed. Check versions

```
$ node -v
v13.5.0 (My Version)
$ npm -v
6.13.4
$npx -v
6.13.4
```

#### Required Packages
```
$ npm install chalk
$ npm install clear
$ npm install figlet
$ npm install commander
$ npm install clui
$ npm install node-fetch
$ npm install inquirer
```
